# KrishiMitra
A multimodal AI project for crop disease detection using image and voice/text.

### 🧠 About
KrishiMitra helps farmers detect crop diseases using **leaf images and audio/text inputs**.  
It gives simple treatment advice in text and audio form, useful for Indian farmers.

---

### ⚙️ Features
- Image-based disease detection  
- Voice or text symptom input (Hindi/English)  
- Text + Speech output for advice  
- Streamlit web interface  

---

### 🧰 Tech Stack
- Python  
- PyTorch / TensorFlow  
- OpenAI Whisper  
- Streamlit  
- gTTS (Text-to-Speech)
